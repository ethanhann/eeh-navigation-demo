metisMenu Integration

eeh-navigation-menu with eeh-metis-menu Directive

The eeh-metis-menu AnglularJS module can be used to add metisMenu menu functionality to the menu generated by the eeh-navigation-menu directive. This is accomplished by adding the eeh-metis-menu attribute to the same element the eeh-navigation-menu directive is applied to. 

<eeh-navigation-menu menu-name="'myMenu'" eeh-metis-menu>
</eeh-navigation-menu>
