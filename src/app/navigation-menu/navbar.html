Bootstrap Navbar and Metis Menu

Menu directives can be mixed and matched. This example pairs metisMenu with eeh-navigation-navbar. The admin dashboard example pairs the eeh-navigation-navbar and eeh-navigation-sidebar directives.

Bootstrap Navbar and Metis Menu Directives

<div class="row">
    <div class="col-lg-12">
        <eeh-navigation-navbar menu-name="'menuOne'"
                                  nav-class="'navbar-inverse navbar-static-top'"
                                  brand-text="'Navbar'"></eeh-navigation-navbar>
    </div>
    <div class="col-sm-3">
        <div class="metis-menu-vertical">
            <eeh-navigation-menu menu-name="'menuTwo'"
                                    eeh-metis-menu class="sidebar-nav"></eeh-navigation-menu>
        </div>
    </div>
</div>
