eeh-navigation-menu

By default the eeh-navigation-menu directive outputs an unstyled, though functional, menu. 

<eeh-navigation-menu menu-name="'menuOne'"></eeh-navigation-menu>

Style eeh-navigation-menu

A tiny snippet of CSS can make the menu look a lot better.

.navigation-menu ul {
    list-style: none;
}

.navigation-menu > ul {
    border: solid 1px #eeeeee;
    background: #dddddd;
    padding: 1em;
}

.navigation-menu ul li > a {
    position: relative;
    display: block;
    padding: 5px 15px;
}

.navigation-menu ul li > a:focus,
.navigation-menu ul li > a:hover {
    text-decoration: none;
    background-color: #eeeeee;
}
