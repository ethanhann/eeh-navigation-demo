eeh-navigation-menu

By default the eeh-navigation-menu directive outputs an unstyled, though functional, menu.

Style eeh-navigation-menu

A tiny snippet of CSS can make the menu look a lot better. Note that the "navigation-menu" CSS class was added to the directive for this example. The unstyled example does not have this CSS class.

.navigation-menu ul {
    list-style: none;
}

.navigation-menu > ul {
    border: solid 1px #eeeeee;
    background: #dddddd;
    padding: 1em;
}

.navigation-menu ul li > a {
    position: relative;
    display: block;
    padding: 5px 15px;
}

.navigation-menu ul li > a:focus,
.navigation-menu ul li > a:hover {
    text-decoration: none;
    background-color: #eeeeee;
}
