Slideout.js Integration

The eeh-slideout AnglularJS module can be used to add Slideout.js touch slideout menu functionality to the menu generated by the eeh-navigation-menu directive. This is accomplished by wrapping the eeh-navigation-menu directive in eeh-slideout directive, along with the site's content.

Where is the example?

This site uses Slideout.js! Click the button on the top right (or swipe right on mobile) to see the menu in action.

<eeh-slideout is-open="slideOutMenuIsOpen" panel-selector="'#panel'" menu-selector="'#menu'"> <nav id="menu"> <eeh-navigation-menu menu-name="'myMenu'"></eeh-navigation-menu> </nav> <main id="panel"> <button ng-click="slideOutMenuIsOpen = !slideOutMenuIsOpen"> toggle slideout menu </button> <section> <p>Main Content</p> </section> </main> </eeh-slideout>
