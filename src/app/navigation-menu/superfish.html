Superfish Integration

The eeh-superfish AnglularJS module can be used to add Superfish menu functionality to the menu generated by the eeh-navigation-menu directive. This is accomplished by adding the eeh-superfish attribute to the same element the eeh-navigation-menu directive is applied to. 

<eeh-navigation-menu menu-name="'myMenu'" eeh-superfish>
</eeh-navigation-menu>
