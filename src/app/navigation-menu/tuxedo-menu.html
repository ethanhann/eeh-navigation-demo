Tuxedo Menu Integration

The eeh-tuxedo-menu AnglularJS module can be used to add Tuxedo Menu functionality to the menu generated by the eeh-navigation-menu directive. This is accomplished by adding the eeh-tuxedo-menu attribute to the same element the eeh-navigation-menu directive is applied to. 

<eeh-navigation-menu menu-name="'myMenu'" eeh-tuxedo-menu></eeh-navigation-menu>
